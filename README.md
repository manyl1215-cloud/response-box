網友回應框產生器 (Social Post Generator)這是一個專為社群媒體創作者設計的網頁工具，能夠快速產生高品質的「網友回應式」圖卡。支援多頁管理、自訂頭像調整、多平台風格切換以及豐富的版面設定。核心功能1. 專案管理 (SLUG)SLUG 命名：在最上方設定專案名稱，下載時會自動以 SLUG_頁碼.png 格式存檔。多頁系統：支援新增、刪除、上一頁與下一頁切換。每一頁的文字、排版與背景設定皆為獨立儲存。2. 文字與版面設定即時換行：文字輸入區支援手動 Enter 換行，或根據框寬度自動計算換行。字型選擇：提供思源黑體、思源明體、手寫風格等多種字體選擇。訊息框調整：可自由拉動滑桿調整白色對話框的寬度與高度。陰影特效：可選擇是否啟用外陰影，並調整陰影的模糊強度（Blur）。3. 頭像系統內建風格：提供「男生、女生、小孩、老人」四種經典正常插畫風格。自訂上傳：支援上傳個人圖片作為頭像，程式會自動裁切為圓形。微調工具：針對自訂頭像提供「縮放大小」、「水平位移」與「垂直位移」滑桿，解決照片人物不在圓心處的問題。4. 社群來源平台支援四種主流平台圖標切換：Facebook：經典藍底 "f" Logo。Instagram：彩色漸層相機圖標。Threads：黑底簡約 "@" 圖標。TikTok：黑底音符風格圖標。5. 背景設定顏色選擇：可透過調色盤選擇任意背景底色。透明度調整：支援 0% 到 100% 透明度，若設為 0% 可產生去背效果的 PNG 檔。使用步驟設定 SLUG：在頂部輸入本次製作的標題。輸入內容：在文字區鍵入想要呈現的話語，並調整字體大小。調整外觀：設定訊息框的尺寸與是否需要陰影。設定頭像：選擇內建人物或上傳照片，並使用位移工具對齊。新增分頁：若有多張圖片需求，點擊「新增頁面」繼續製作。下載成果：點擊「下載 PNG 圖片」即可獲得解析度 1200x630 的社群標準格式圖片。技術規格技術棧：HTML5, Canvas API, Tailwind CSS, Google Fonts.輸出解析度：1200 x 630 px.儲存格式：PNG (支援透明度).
